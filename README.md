# Photorina

### A C++ based photo sorter.  
_____________________________________

Open-source automated file mover specialized for organizing image file formats. 
- Create folders/directories directly while moving files
- Automate sorting with filters for camera, date, and other metadata.

Dependencies used: tomlc++

_____________________________________

Installation:
Download the executable on the downloads page (WIP)

From build:
- Use your C++ compiler of choice to build `photosorter.cpp` and `photoSort.h` together. Ensure that `toml.hpp` is included for the parsing library.
